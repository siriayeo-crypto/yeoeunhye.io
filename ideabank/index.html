📊 Google Sheets 연동

Apps Script 웹앱 URL을 입력하면 Sheets와 양방향으로 동기화돼요.

1
Google Sheets 새 파일 생성
2
확장 프로그램 → Apps Script → 코드 붙여넣기
3
배포 → 웹앱으로 배포 → 누구나 접근 허용
4
배포 URL을 아래에 붙여넣기

※ URL을 비워두면 연동이 해제돼요.
